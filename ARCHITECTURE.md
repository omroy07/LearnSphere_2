# Architecture Documentation

## Tech Stack
- **Frontend Framework:** HTML, CSS, JavaScript (Vanilla/ES6+)
- **Build Tool:** Vite
- **Linting & Formatting:** ESLint, Prettier
- **Testing:** Vitest

## Main Modules and Folders
- **Root Files:** Contains various JavaScript modules (e.g., explore.js, progress.js, 
otifications.js, etc.) acting as independent entry points.
- **sub/:** Contains styles for sub-modules.
- **quiz/:** Contains core quiz-related functionality, adaptive quiz logic, and its UI styles.
- **static/:** General static assets.
- **dist/:** Output directory generated by Vite.

## Data Flow
- **State Management:** The application relies heavily on local state and localStorage to sync data between the user sessions.
- **Offline Sync:** Uses Service Workers (sw.js and offlineSync.js) to provide offline capabilities and synchronize user progress once they reconnect.
- **Module Communication:** Each page incorporates specific modules through <script> tags, using global window variables or events to communicate status updates (e.g., in courses.html).
